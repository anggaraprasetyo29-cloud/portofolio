Sistem Informasi Manajemen & Distribusi Terintegrasi (Syarif Durian)

Studi kasus capstone: otomasi pergudangan + SPK (SAW) untuk ranking prioritas agen.

Deskripsi

Sistem informasi berbasis web untuk mengotomatisasi proses bisnis pergudangan dan distribusi. Mengintegrasikan modul Sistem Pendukung Keputusan (SPK) menggunakan metode Simple Additive Weighting (SAW) untuk meranking prioritas agen secara presisi.

Kontribusi Utama

  • System Analyst: observasi & wawancara, perumusan kriteria (contoh: “Waktu Bayar”).
  • System Analyst: perancangan logika SAW (cost/benefit, pembobotan).
  • System Analyst: pemodelan alur dengan Prototype (alur searah).
  • Web Developer: implementasi aplikasi web PHP & MySQL.
  • Web Developer: desain basis data relasional + manajemen user (MD5).
  • Web Developer: deployment, migrasi DB, konfigurasi hosting/cPanel.
